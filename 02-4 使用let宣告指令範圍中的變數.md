# 02-4 使用let宣告指令範圍中的變數

```
用let宣告某指令範圍(如if, for)的變數; 如果函式中的某指令範疇內宣告var, 指的是宣告了函式範圍的變數.
```

### 在if中用let宣告指令範圍內的區域變數

```javascript
//--------------------------------------
// 使用嚴格模式, 使用的變數一定要事前宣告
//--------------------------------------
'use strict';

//----------------------------------------- 
// 一個計算成績點數的函式
//----------------------------------------- 
function points(chi, eng, mat){
    var p=0;  //在函式中使用的區域變數

    if(chi >= 60){
        let m = parseInt(chi / 10);  //在此if中使用的區域變數
        p+=m;
    }
    
    if(eng >= 60){
        let m = parseInt(eng / 15);  //在此if中使用的區域變數  
        p+=m;        
    }
    
    if(mat >= 60){
        let m = parseInt(mat / 12);  //在此if中使用的區域變數  
        p+=m;         
    }

    return p;
}

//------------------------------
// 主程式
//------------------------------
console.log(points(60, 50, 77));    //12
console.log(points(55, 64, 95));    //11
```
