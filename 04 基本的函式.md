# 04 基本的函式

### (1) 一個基本的函式範例

```javascript
//-------------------------
// 一個執行相加的函式
//-------------------------
function add(a, b=15){
    return a + b;
}

//-------------------------
// 主程式
//-------------------------
console.log(add(10, 20));   //30
console.log(add(10))        //25
console.log(add())          //NaN
```


### (2) 某個變數的內容是一個匿名函式

```javascript
//-------------------------
// 將變數f指向一個函式
//-------------------------
var f = function(a, b=15){
    return a + b;
}

//-------------------------
// 主程式
//-------------------------
console.log(f(10, 20));   //30
```



### (3) 直接執行匿名函式

```javascript
//-------------------------
// 直接執行匿名函式
//-------------------------
var m = function(a, b=15){
    return a + b;
}(10, 20);

//-------------------------
// 主程式
//-------------------------
console.log(m);   //30
```


### (4) 將函式傳給另一個函式

```javascript
//------------------------------
// 一個執行相加的函式
//------------------------------
function add(a, b=15, adder){
    return adder(a, b);
}

//------------------------------
// 定義相加功能的函式
//------------------------------
function adder(a, b){
    return a + b;
}

//------------------------------
// 主程式
//------------------------------
console.log(add(10, 20, adder));   //30
```


### (5) 使用箭頭函式
```
資料來源: https://developer.mozilla.org/zh-TW/docs/Web/JavaScript/Reference/Functions/Arrow_functions

箭頭函式運算式（arrow function expression）擁有比函式運算式還簡短的語法。
它沒有自己的 this、arguments、super、new.target 等語法。
本函式運算式適用於非方法的函式，但不能被用作建構式（constructor）。
```

```javascript
//------------------------------
// 一個執行相加的函式
//------------------------------
function add(a, b=15, adder){
    return adder(a, b);
}

//------------------------------
// 主程式
//------------------------------
console.log(add(10, 20, (x, y) => {return x + y}));   //30
```
