# 08-6 閉包

```
一個閉包包括一個函式及該函式生成時建立於函式之外的變數,
當函式物件存在時, 函式外引用的變數也會存在, 不會被回收. 
```

### 程式範例

```javascript
//-------------------------
// 使用嚴格模式
//-------------------------
'use strict';

//-------------------------
// 閉包:一個函式及變數x
//-------------------------
function counter(){
    var x = 0;  //閉包函式使用的變數

    //一個閉包的函式
    return () => { 
        x++;
        return x;
    }
}

//-------------------------
// 測試
//-------------------------
var a = counter();
console.log(a());   //1
console.log(a());   //2

var b = counter();
console.log(b());   //1
console.log(b());   //2
console.log(b());   //3
```
