# 09-3 靜態方法

```
使用靜態方法不必先產生物件, 直接呼叫方法即可.
```

### 程式範例, 繼承

```javascript
//---------------------------------------
// 使用嚴格模式
//---------------------------------------
'use strict';

//---------------------------------------
// 一個類別
//---------------------------------------
class Utility{
    //==================================================
    // 一個計算bmi的靜態方法
    // 傳入: 身高(cm), 體重(kg)
    // 回傳: bmi
    //==================================================    
    static bmi(height, weight){
        return weight / Math.pow(height/100, 2);     
    }

    
    //==================================================
    // 一個評估bmi的靜態方法
    // 傳入: 身高(cm), 體重(kg)
    // 回傳: bmi評語
    //==================================================  
    static evaluation(height, weight){
        var bmi = Utility.bmi(height, weight);
        
        if(bmi < 18.5){
            return "過輕";
        }else if(bmi >= 18.5 && bmi < 24){
            return "正常體重";
        }else if(bmi >= 24 && bmi < 27){
            return "過重";
        }else{
            return "肥胖";
        }
    }
    //==================================================   
}

//---------------------------------------
// 測試
//---------------------------------------
console.log('bmi:' + Utility.bmi(170, 60));
console.log('評語:' + Utility.evaluation(170, 60));

console.log('bmi:' + Utility.bmi(160, 55));
console.log('評語:' + Utility.evaluation(160, 55));

console.log('bmi:' + Utility.bmi(180, 85));
console.log('評語:' + Utility.evaluation(180, 85));
```
