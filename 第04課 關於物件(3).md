#第04課 關於物件(3)

## (1) 函式屬性

#####測試程式
```javascript
//====================
// 宣告一個物件
//====================
var m={
    stuNo:'1456001',
    name:'王小明',
    gender:'M',

    getStuNo:function(){return this.stuNo;},
    getName:function(){return this.name;},
    
    //-----------------
    // 回傳中文性別
    //-----------------
    getGender:function(){
        var gender='有誤';
        
        if(this.gender=='M'){
            gender='男';
        }else{
            gender='女';
        }
        
        return gender;
    }
}


//====================
// 呼叫物件中的方法
//====================
console.log(m.getStuNo());
console.log(m.getName());
console.log(m.getGender());
```



#####執行結果
```
1456001
王小明
男
```



## (2) 呼叫函式屬性的call(...)方法

#####測試程式
```javascript
//====================
// 宣告一個物件
//====================
var m={
    stuNo:'1456001',
    name:'王小明',
    gender:'M',

    getStuNo:function(){return this.stuNo;},
    getName:function(){return this.name;},
    
    //-----------------
    // 回傳中文性別
    //-----------------
    getGender:function(){
        var gender='有誤';
        
        if(this.gender=='M'){
            gender='男';
        }else{
            gender='女';
        }
        
        return gender;
    }
}


//====================
// 呼叫物件中的方法
//====================
console.log(m.getStuNo());
console.log(m.getName());

//-----------------------------------------
// 使用call(...)方法, 呼叫某物件中的函式
//-----------------------------------------
var parameter={gender:'F'};
console.log(m.getGender.call(parameter));
```



#####執行結果
```
1456001
王小明
女
```
