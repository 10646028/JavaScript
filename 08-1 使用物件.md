# 08-1 使用物件



### (1) 程式範例, 建立物件

```javascript
//----------------------
// 使用嚴格模式
//----------------------
'use strict';

//----------------------
// 印出物件內容
//----------------------
function printInfo(s){
    console.log(s);

    console.log(s.stuNo);
    console.log(s['stuNo']);
    
    console.log(s.stuName);
    console.log(s['stuName']);
}

//----------------------
// 建立一個物件
//----------------------
var stuNo = '120001';
var stuName = '王小明';
var s1 = {stuNo, stuName};

printInfo(s1);

//----------------------
// 建立一個物件
//----------------------
var s2 = {
    'stuNo':'120001',
    'stuName':'王小明'
}

printInfo(s2);
```
