#第03課 關於物件



## (1) 物件中的方法

#####測試程式
```javascript
//====================
// 宣告一個物件
//====================
var m={
	stuNo:'1456001',
    name:'王小明',
    gender:'M',
    
    // this表示目前物件中的屬屬性
    getStuNo:function(){return this.stuNo;},
    getName:function(){return this.name;},
    getGender:function(){return this.gender;}
}


//====================
// 呼叫物件中的方法
//====================
console.log(m.getStuNo());
console.log(m.getName());
console.log(m.getGender());
```



#####執行結果
```
1456001
王小明
M
```



## (2) 取用屬性與區域/廣域變數的區別


#####測試程式
```javascript
//====================
// 宣告一個物件
//====================
var m={
	stuNo:'1456001',
    name:'王小明',
    gender:'M',
    
    // 不寫this時, 表示要依序找出(1)區域變數stuNo (2)廣域變數stuNo
    // 如果都找不到, 則彈出錯誤訊息.
    getStuNo:function(){return stuNo;},  
    getName:function(){return this.name;},
    getGender:function(){return this.gender;}
}


//====================
// 呼叫物件中的方法
//====================
console.log(m.getStuNo());
console.log(m.getName());
console.log(m.getGender());
```



#####執行結果 (執行錯誤訊息)
```
ReferenceError: stuNo is not defined
    at Object.getStuNo:10:34
    at eval:19:15
    at eval
```
