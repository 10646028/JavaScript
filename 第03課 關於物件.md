#第03課 關於物件



## (1) 物件中的方法

#####測試程式
```javascript
//====================
// 宣告一個物件
//====================
var m={
    stuNo:'1456001',
    name:'王小明',
    gender:'M',
 
    //------------------------------------------------------
    // return this.stuNo; 表示回傳所在物件中的stuNo屬性內容
    //------------------------------------------------------
    getStuNo:function(){return this.stuNo;},
    getName:function(){return this.name;},
    getGender:function(){return this.gender;}
}


//====================
// 呼叫物件中的方法
//====================
console.log(m.getStuNo());
console.log(m.getName());
console.log(m.getGender());
```



#####執行結果
```
1456001
王小明
M
```



## (2) 取用物件屬性與區域/廣域變數的區別


#####測試程式
```javascript
//====================
// 宣告一個物件
//====================
var m={
    stuNo:'1456001',
    name:'王小明',
    gender:'M',
    
    //------------------------------------------------
    // return stuNo; 指令的stuNo不加註this.限定對象,
    // 則依序尋找(1)區域變數stuNo (2)廣域變數stuNo 的值回傳,
    // 如果都找不到, 則彈出錯誤訊息.
    //------------------------------------------------
    getStuNo:function(){return stuNo;},  
    getName:function(){return this.name;},
    getGender:function(){return this.gender;}
}


//====================
// 呼叫物件中的方法
//====================
console.log(m.getStuNo());
console.log(m.getName());
console.log(m.getGender());
```



#####執行結果 (錯誤訊息)
```
ReferenceError: stuNo is not defined
    at Object.getStuNo:10:34
    at eval:19:15
    at eval
```



## (3) 取用區域/廣域變數


#####測試程式
```javascript
//=============================
// 宣告一個廣域變數stuNo
//=============================
var stuNo='這是廣域變數stuNo';


//====================
// 宣告一個物件
//====================
var m={
    stuNo:'1456001',
    name:'王小明',
    gender:'M',
    
    getStuNo:function(){
    	//-------------------------
        // 宣告一個區域變數stuNo
    	//-------------------------   
        var stuNo='這是區域變數stuNo';
    	return stuNo;
    },
    getName:function(){return this.name;},
    getGender:function(){return this.gender;}
}


//====================
// 呼叫物件中的方法
//====================
console.log(m.getStuNo());
console.log(m.getName());
console.log(m.getGender());
```



#####執行結果
```
這是區域變數stuNo
王小明
M
```
