# 09 箭頭函式

### (1) 範例

```
以下的箭頭函式都與以下相同:
function f(x){
    return x*x;
}

箭頭函式的參數部份, 如果只有1個參數, 可以不加小括號;
箭頭函式的內容部份, 如果不加大括號, 回傳值可不寫return.
```

```javascript
//--------------------------------------
// 使用嚴格模式, 使用的變數一定要事前宣告
//--------------------------------------
'use strict';

//----------------------------------------- 
// 宣告一個箭頭函式
//----------------------------------------- 
var f1 = x => x*x;

//----------------------------------------- 
// 宣告一個箭頭函式
//----------------------------------------- 
var f2 = x => {return x*x};

//----------------------------------------- 
// 宣告一個箭頭函式
//----------------------------------------- 
var f3 = (x) => x*x;

//----------------------------------------- 
// 宣告一個箭頭函式
//----------------------------------------- 
var f4 = (x) => {return x*x};

//------------------------------
// 主程式
//------------------------------
console.log(f1(10));    //100
console.log(f2(10));    //100
console.log(f3(10));    //100
console.log(f4(10));    //100
```


### (2) 傳2個參數

```javascript
//--------------------------------------
// 使用嚴格模式, 使用的變數一定要事前宣告
//--------------------------------------
'use strict';

//----------------------------------------- 
// 宣告一個箭頭函式
//----------------------------------------- 
var f = (x, y) => {if(x>y) return x*x; else return y*y;};

//------------------------------
// 主程式
//------------------------------
console.log(f(10, 20));    //400
console.log(f(25, 20));    //625
```
