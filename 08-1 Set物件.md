# 08-1 Set物件

```
Set 物件存放的資料不會重覆.
```


### 程式範例

```javascript
//------------------------------------------
// 使用嚴格模式
//------------------------------------------
'use strict';

//------------------------------------------
// 建立一個Set物件 (存放物件不重覆)
//------------------------------------------
const set = new Set();

//------------------------------------------
// 加入資料
//------------------------------------------
set.add(10);
set.add(20);
set.add('Hello');
set.add(10);

//------------------------------------------
// 印出資料
//------------------------------------------
console.log('資料個數:' + set.size);

set.forEach(x => { 
    console.log(x);
});

//------------------------------------------
// 刪除資料
//------------------------------------------
set.delete(10);
set.delete(30);

//------------------------------------------
// 印出資料
//------------------------------------------
console.log('資料個數:' + set.size);

set.forEach(x => { 
    console.log(x);
});

//------------------------------------------
// 判斷Set中是否存在某資料?
//------------------------------------------
console.log('set中是否有10?' + set.has(10));
console.log('set中是否有20?' + set.has(20));
console.log('set中是否有30?' + set.has(30));
```
