# 08-1 Set物件

```
Set 物件存放的資料不會重覆.
```


### 程式範例

```javascript
//------------------------------------------
// 使用嚴格模式
//------------------------------------------
'use strict';

//------------------------------------------
// 建立一個Set物件 (的存放物件不重覆)
//------------------------------------------
var set = new Set();

//------------------------------------------
// 加入資料
//------------------------------------------
set.add(10);
set.add(20);
set.add('Hello');
set.add(10);

//------------------------------------------
// 印出資料
//------------------------------------------
console.log('資料個數:' + set.size);

set.forEach(x => { 
    console.log(x);
});

//------------------------------------------
// 刪除資料
//------------------------------------------
set.delete(10);
set.delete(30);

//------------------------------------------
// 印出資料
//------------------------------------------
console.log('資料個數:' + set.size);

set.forEach(x => { 
    console.log(x);
});
```
